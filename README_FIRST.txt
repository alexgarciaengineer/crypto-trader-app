To run locally:

1. Open terminal and navigate to this folder:
   cd path/to/crypto-trader-app

2. Install dependencies:
   pip install -r requirements.txt

3. Start the app:
   python app.py

Then open your browser to:
   http://127.0.0.1:5000

The app will prompt you to enter your Coinbase API key and secret.
Your summary dashboard will appear once connected.
